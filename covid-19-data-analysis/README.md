COVID-19 Data Analysis Project

Project Overview:
This project analyzes COVID-19 data to understand the spread of the virus, confirmed cases, recoveries, 
and deaths across different regions and time periods.

Tools & Technologies:
Python
Pandas
Matplotlib / Seaborn
Jupyter Notebook

Key Analysis Performed:
Data cleaning 
Exploratory Data Analysis (EDA)
Univariate and Bivariate Analysis
Country/region-wise comparison

Dataset:
Source: Kaggle
Format: CSV

Results & Insights:
The impact of COVID-19 was uneven, with a small number of countries accounting for most global cases, deaths, and active infections.
The United States emerged as the epicenter, reporting the highest number of new cases, active cases (65.9%), and deaths (41%) among top affected countries.
Countries with higher confirmed cases also recorded higher deaths, while recoveries were significantly greater than deaths overall.
